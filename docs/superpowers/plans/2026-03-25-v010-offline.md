# Socratiq v0.1.0 Offline — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn a B站 programming tutorial playlist into an interactive course with structured lessons, Mermaid diagrams, code labs with tests, and an AI tutor — launchable via `docker compose up`.

**Architecture:** Extend the existing ingestion pipeline (extract → analyze → store → embed) with two new stages: LessonGenerator (subtitles → structured lesson with diagrams/steps/code) and LabGenerator (code snippets → skeleton + tests + run instructions). Add Bilibili playlist support, Docker containerization, and an onboarding flow. Rewrite the frontend learn page as a course reader with lesson/video/lab/tutor tabs.

**Tech Stack:** Existing stack (FastAPI, Next.js 16, PostgreSQL, Redis, Celery) + mermaid (frontend), Docker multi-stage builds

**Branch:** `offline`

**Spec:** `docs/superpowers/specs/2026-03-25-v010-offline-design.md`

```python
from pydantic import BaseModel

class CodeSnippet(BaseModel):
    language: str
    code: str
    context: str

class Diagram(BaseModel):
    type: str       # "mermaid" | "comparison"
    title: str
    content: str    # Mermaid syntax or JSON

class Step(BaseModel):
    label: str
    detail: str
    code: str | None = None

class StepByStep(BaseModel):
    title: str
    steps: list[Step]

class LessonSection(BaseModel):
    heading: str
    content: str
    timestamp: float
    code_snippets: list[CodeSnippet] = []
    key_concepts: list[str] = []
    diagrams: list[Diagram] = []
    interactive_steps: StepByStep | None = None

class LessonContent(BaseModel):
    title: str
    summary: str
    sections: list[LessonSection]
```
